<h1>개발 목표</h1>

· 리액트를 기본 문법 및 state 사용 방법, 컴포넌트를 이해하고 활용하는데 중점 <br>
· 사용자가 선택한 지역의 실시간 날씨정보를 표시하는 기능 구현


<h1>사용기술</h1>

· react<br>
· bootstrap<br>
· css<br>
· es6<br>

<h1>개선사항</h1>

· 앱의 반응형 디자인을 구현하여 다양한 기기와 화면 크기에 대응 필요<br>
· 검색 기능을 추가하여 사용자가 원하는 지역의 날씨 정보를 조회할 수 있도록 개선 필요<br>
